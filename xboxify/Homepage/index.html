XboxIfy

With XboxIfy, you can rename files on your (local) harddisk, so that they can be easily transferred to the Xbox.
XboxIfy should be very easy to use. After starting the program, you only have to select the folder containing the files and click on "Start"! XboxIfy will then walk through all of the directories and subdirectories to find all the files. It will then remove every "Funny" character it can find. "Funny" characters are those characters, which the FATX filesystem of the Xbox harddisks are not allowed to contain (like , (comma), ;, ? ...). XboxIfy will also shorten the filename to 42 characters. Besides cleaning filenames, it will also make sure that directorynames are "safe".

XboxIfy is written in Python with the wxPython/wxWindows toolkit. Because of this, it works on Windows, Linux and MacOS X (however, since I don't have a Mac, I can only test on Linux and Windows. Donations are more than welcome *G*). For Windows, binaries containing everything needed are provided.

XboxIfy is far from perfect and there are for sure quite some bugs in it - if you find any, please feel free to contact me at XboxIfy@message-center.info! Feature suggestions are also welcome.
